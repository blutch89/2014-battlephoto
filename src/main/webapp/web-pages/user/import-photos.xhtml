Importation de photos
Marche à suivre pour ajouter des photos

1. Commencez par cliquer sur le bouton "Choose" et sélectionner les photos sur votre ordinateur.

2. Une fois les photos sélectionnées, cliquez sur "Upload" pour ajouter les photos.

3. Une fois les photos ajoutées, cliquez sur le bouton "Suivant" pour modifier les informations des photos.
