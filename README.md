# Description
Application "carte de visite" de concours photo développé avec Java EE.

# Informations:
* Cette application fonctionne avec Maven. Il ne faut donc pas oublier d'importer les librairies.

# Technologies:
* Java 7
* Maven
* Spring, Spring Security
* AspectJ
* Hibernate
* JSF, Primefaces
* PrettyFaces
* Log4J

# Serveur d'application:
* Apache Tomcat 8

# IDE:
* Eclipse
